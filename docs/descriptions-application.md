**TeranGuest – Application client**

TeranGuest est l’application d’accès client des établissements partenaires. Elle permet aux clients de :

• Accéder à leur séjour via un code client ou un QR code  
• Consulter les services de l’établissement (restaurant, spa, excursions, blanchisserie, etc.)  
• Passer des commandes (room service, réservations restaurant, spa, excursions)  
• Suivre leurs commandes et réservations  

L’application est sécurisée et respecte la confidentialité des données personnelles.
