## 🚀 PARCOURS UTILISATEUR

### Parcours 1 : Commander un Repas
```
Dashboard
  → Room Service
    → Catégories (4 col 3D)
      → Articles (4 col 3D)
        → Détail Article
          → Ajouter au panier
            → Panier (badge mis à jour)
              → Commander
                → Confirmation ✅
                  → Profil → Mes Commandes
```

### Parcours 2 : Réserver une Table
```
Dashboard
  → Restaurants & Bars
    → Liste (4 col 3D, filtres)
      → Détail Restaurant
        → Réserver
          → Formulaire (date, heure, personnes)
            → Confirmation ✅
              → Profil → Mes Réservations Restaurant
```

### Parcours 3 : Réserver un Soin Spa
```
Dashboard
  → Spa & Bien-être
    → Liste Services (4 col 3D, filtres)
      → Détail Service
        → Réserver
          → Formulaire (date, heure)
            → Confirmation ✅
              → Profil → Mes Réservations Spa
```

### Parcours 4 : Réserver une Excursion
```
Dashboard
  → Excursions
    → Liste (4 col 3D)
      → Détail Excursion
        → Réserver
          → Formulaire (date, adultes, enfants)
            → Confirmation ✅
              → Profil → Mes Excursions
```

### Parcours 5 : Demande Blanchisserie
```
Dashboard
  → Blanchisserie
    → Liste Services (4 col 3D)
      → Sélectionner quantités (+ / -)
        → Footer récapitulatif (total)
          → Confirmer
            → Formulaire (instructions)
              → Confirmation ✅
                → Profil → Mes Demandes Blanchisserie
```

### Parcours 6 : Demande Service Palace
```
Dashboard
  → Services Palace
    → Liste Services (4 col 3D)
      → Sélectionner Service
        → Formulaire (détails, date/heure optionnelle)
          → Envoyer
            → Confirmation ✅
              → Profil → Mes Demandes Palace
```
